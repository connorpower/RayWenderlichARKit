# ARFaceCase

Chapter 14-18 exercise from Ray Wenderlich's 'ARKit by Tutorials' book, v3.0.0.
This project allows the user to overlay various masks and SceneKit objects on
their face in a similar manner to Snapchat or Apple Animoji.

Rendering is performed by SceneKit (not the newer RealityKit).

## Environment

- ARKit by Tutorials, 3.0.0
- Xcode 11.3
- iOS 13.2
- ARKit 3 with SceneKit
